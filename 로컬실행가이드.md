# 로컬에서 Streamlit 앱 실행하기

## 방법 1: 배치 파일 사용 (가장 간단) ⭐ 추천

1. **프로젝트 폴더로 이동**
   - 파일 탐색기에서 다음 경로로 이동:
     ```
     C:\Users\USER-PC\OneDrive\바탕 화면\CURORSTUDY\store_ops_v1
     ```

2. **run.bat 파일 더블클릭**
   - `run.bat` 파일을 더블클릭하면 자동으로:
     - 가상환경 생성 (없는 경우)
     - 필요한 패키지 설치
     - Streamlit 앱 실행

3. **브라우저 자동 실행**
   - 자동으로 브라우저가 열리고 `http://localhost:8501` 주소로 접속됩니다

---

## 방법 2: 명령 프롬프트(CMD) 사용

1. **명령 프롬프트 열기**
   - `Win + R` 키 누르기
   - `cmd` 입력 후 Enter

2. **프로젝트 폴더로 이동**
   ```cmd
   cd "C:\Users\USER-PC\OneDrive\바탕 화면\CURORSTUDY\store_ops_v1"
   ```

3. **가상환경 생성 (처음 한 번만)**
   ```cmd
   python -m venv .venv
   ```

4. **가상환경 활성화**
   ```cmd
   .venv\Scripts\activate
   ```

5. **필수 패키지 설치 (처음 한 번만)**
   ```cmd
   pip install -r requirements.txt
   ```

6. **Streamlit 실행**
   ```cmd
   streamlit run app.py
   ```

7. **브라우저에서 접속**
   - 자동으로 브라우저가 열리거나
   - 수동으로 `http://localhost:8501` 접속

---

## 방법 3: PowerShell 사용

1. **PowerShell 열기**
   - `Win + X` 키 누르기
   - "Windows PowerShell" 또는 "터미널" 선택

2. **프로젝트 폴더로 이동**
   ```powershell
   cd "C:\Users\USER-PC\OneDrive\바탕 화면\CURORSTUDY\store_ops_v1"
   ```

3. **가상환경 생성 및 활성화**
   ```powershell
   python -m venv .venv
   .venv\Scripts\Activate.ps1
   ```

4. **필수 패키지 설치**
   ```powershell
   pip install -r requirements.txt
   ```

5. **Streamlit 실행**
   ```powershell
   streamlit run app.py
   ```

## 브라우저에서 툴팁 제거하기

로컬에서 실행한 후, 브라우저 개발자 도구를 사용하여 툴팁을 제거할 수 있습니다:

### Chrome/Edge 브라우저:

1. **개발자 도구 열기**
   - `F12` 키 누르기
   - 또는 `Ctrl + Shift + I`

2. **Console 탭 클릭**

3. **다음 코드 붙여넣기 후 Enter**
   ```javascript
   // 모든 keyboard 관련 title 제거
   setInterval(() => {
       document.querySelectorAll('*').forEach(el => {
           if (el.title && el.title.toLowerCase().includes('keyboard')) {
               el.removeAttribute('title');
           }
       });
   }, 100);
   
   // title 속성 설정 차단
   const originalSetAttribute = Element.prototype.setAttribute;
   Element.prototype.setAttribute = function(name, value) {
       if (name === 'title' && typeof value === 'string' && value.toLowerCase().includes('keyboard')) {
           return;
       }
       return originalSetAttribute.call(this, name, value);
   };
   ```

4. **사이드바 토글 버튼에 마우스 올려보기**
   - 이제 툴팁이 표시되지 않아야 합니다

## 문제 해결

### Python이 설치되지 않은 경우
- [Python 공식 사이트](https://www.python.org/downloads/)에서 Python 3.8 이상 버전 다운로드 및 설치
- 설치 시 "Add Python to PATH" 옵션 체크

### 포트가 이미 사용 중인 경우
- 다른 터미널에서 실행 중인 Streamlit 앱 종료
- 또는 다른 포트 사용: `streamlit run app.py --server.port 8502`

### 패키지 설치 오류
```cmd
pip install --upgrade pip
pip install -r requirements.txt
```
