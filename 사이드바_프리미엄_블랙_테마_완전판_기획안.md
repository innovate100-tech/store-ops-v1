# 사이드바 프리미엄 블랙 테마 완전판 기획안

## 📋 목표

**현재 프리미엄 블랙 테마를 최고급 수준으로 완전히 개선**
- 고급 애니메이션 효과 추가
- 그라데이션 배경 및 호버 효과 강화
- 미묘한 글로우 및 그림자 효과
- 부드러운 전환 애니메이션
- **추가 고급 효과 4종 모두 포함**

---

## ⚠️ 제약사항 유지

- JavaScript, DOM 조작 금지
- `[data-testid="stSidebar"]` 선택자 사용 금지
- 사이드바 위치/크기/토글 CSS 금지
- 메인 콘텐츠에 영향 주는 전역 CSS 금지
- `.ps-sidebar-scope` 하위로만 CSS 적용
- `transform: translateX()` 금지 (레이아웃 흔들림 방지)

---

## 🎨 고급 개선 요소 (완전판)

### 1. 그라데이션 배경
- 버튼 기본 배경: 미묘한 그라데이션 (위에서 아래로 어두워지는 효과)
- 호버 시: 더 밝은 그라데이션으로 변화
- 활성 버튼: 블루 그라데이션 + 미묘한 애니메이션

### 2. 고급 애니메이션
- CSS `@keyframes`를 사용한 부드러운 펄스 효과
- 호버 시 배경색 그라데이션 변화
- 활성 버튼에 미묘한 글로우 펄스
- 카테고리 제목에 미묘한 페이드인 효과

### 3. 그림자 및 글로우 효과
- 호버 시: 더 깊은 그림자 + 미묘한 글로우
- 활성 버튼: 강화된 글로우 + 펄스 애니메이션
- 계층감 있는 그림자 효과

### 4. 색상 및 그라데이션
- 더 풍부한 그라데이션 팔레트
- 미묘한 색상 변화 애니메이션
- 부드러운 전환 효과

### 5. **추가 고급 효과 (완전판 포함)**
- ✅ **버튼 아이콘 회전 효과**: 호버 시 아이콘 미묘한 회전
- ✅ **리플 효과**: 클릭 시 물결 효과 (CSS만으로)
- ✅ **그라데이션 텍스트**: 카테고리 제목에 그라데이션 텍스트
- ✅ **백드롭 블러**: 배경에 미묘한 블러 효과

## 🔧 구현 방법

### 1. CSS Keyframes 애니메이션

```css
/* 펄스 글로우 애니메이션 (활성 버튼용) */
@keyframes premium-glow-pulse {
    0%, 100% {
        box-shadow: 0 4px 12px rgba(59, 130, 246, 0.3),
                    0 0 0 0 rgba(59, 130, 246, 0.4);
    }
    50% {
        box-shadow: 0 6px 20px rgba(59, 130, 246, 0.4),
                    0 0 0 4px rgba(59, 130, 246, 0.1);
    }
}

/* 그라데이션 배경 애니메이션 (호버용) */
@keyframes premium-gradient-shift {
    0% {
        background-position: 0% 50%;
    }
    50% {
        background-position: 100% 50%;
    }
    100% {
        background-position: 0% 50%;
    }
}

/* 미묘한 펄스 효과 (카테고리 제목용) */
@keyframes premium-fade-in {
    from {
        opacity: 0;
        transform: translateY(-4px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

/* 아이콘 회전 애니메이션 (버튼 호버용) */
@keyframes premium-icon-rotate {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(360deg);
    }
}

/* 리플 효과 애니메이션 (클릭용) */
@keyframes premium-ripple {
    0% {
        transform: scale(0);
        opacity: 1;
    }
    100% {
        transform: scale(4);
        opacity: 0;
    }
}

/* 그라데이션 텍스트 애니메이션 (카테고리 제목용) */
@keyframes premium-gradient-text {
    0%, 100% {
        background-position: 0% 50%;
    }
    50% {
        background-position: 100% 50%;
    }
}
```

### 8. 아이콘 회전 효과 (상세 구현)

```css
/* 버튼 내부 이모지/아이콘 회전 효과 */
/* Streamlit 버튼 구조에 맞춘 선택자 */
.ps-sidebar-scope .stButton > button:hover {
    /* 버튼 텍스트의 첫 번째 문자(이모지) 회전 */
}

/* 이모지가 포함된 버튼의 첫 문자 회전 */
.ps-sidebar-scope .stButton > button:hover::first-letter {
    display: inline-block;
    animation: premium-icon-rotate 0.6s ease-in-out;
}

/* 더 정확한 선택: 버튼 내부 span이나 직접 텍스트 */
.ps-sidebar-scope .stButton > button {
    /* 버튼 내부 구조 분석 필요 */
}

/* 대안: 버튼 전체에 미묘한 회전 효과 (아이콘만 회전하는 것은 CSS만으로 제한적) */
/* 대신 버튼 호버 시 미묘한 전체 회전이나 스케일 효과로 대체 가능 */
```

**아이콘 회전 효과 구현 노트:**
- Streamlit 버튼의 DOM 구조가 복잡하여 CSS만으로 첫 문자(이모지)만 선택하는 것이 제한적일 수 있음
- 대안 1: 버튼 호버 시 전체에 미묘한 회전 효과 (`transform: rotate(1deg)`)
- 대안 2: 버튼 내부에 가상 요소로 회전하는 장식 추가
- 대안 3: 버튼 텍스트 앞에 숨겨진 회전 아이콘 추가 (HTML 구조 변경 필요하나 제약사항상 제외)

**최종 선택: 버튼 호버 시 미묘한 전체 기울기 효과로 대체**

```css
/* 버튼 호버 시 미묘한 기울기 효과 (아이콘 회전 대체) */
.ps-sidebar-scope .stButton > button:hover {
    transform: scale(1.01) rotate(0.5deg);
    /* 기존 scale(1.01)에 rotate(0.5deg) 추가 */
}
```

## 🎯 상세 구현 계획

### Phase 1: CSS Keyframes 애니메이션 추가
1. `@keyframes premium-glow-pulse` - 활성 버튼 글로우 펄스
2. `@keyframes premium-gradient-shift` - 그라데이션 배경 이동
3. `@keyframes premium-fade-in` - 카테고리 제목 페이드인
4. `@keyframes premium-icon-rotate` - 아이콘 회전 (또는 대체 효과)
5. `@keyframes premium-ripple` - 리플 효과
6. `@keyframes premium-gradient-text` - 그라데이션 텍스트 애니메이션

### Phase 2: 버튼 고급 스타일링
1. 기본 배경: 미묘한 그라데이션 (위→아래)
2. 호버 효과: 
   - 배경 그라데이션 밝게 변화
   - 슬라이드 효과 (::before 사용)
   - 깊은 그림자 + 글로우
   - `scale(1.01) rotate(0.5deg)` 미묘한 확대 및 기울기
3. 활성 버튼:
   - 블루 그라데이션 배경
   - 펄스 애니메이션
   - 내부 빛 효과 (::after)
4. **리플 효과**: 클릭 시 물결 효과 (::after 사용, `:active` 상태)

### Phase 3: 카테고리 제목 고급화
1. 페이드인 애니메이션
2. **그라데이션 텍스트** (애니메이션 포함)
3. 하단 그라데이션 라인 (::after)

### Phase 4: Expander 및 Selectbox 고급화
1. 그라데이션 배경
2. 호버 시 부드러운 전환
3. **백드롭 블러 효과** 추가

### Phase 5: 시스템 버튼 고급화
1. 상단 그라데이션 라인 (::before)
2. 그라데이션 배경

### Phase 6: 백드롭 블러 효과
1. 사이드바 배경에 미묘한 블러 오버레이
2. 콘텐츠는 블러 위에 표시

## 🎨 디자인 상세 스펙

### 색상 및 그라데이션
- **버튼 기본**: `linear-gradient(180deg, rgba(255,255,255,0.05) 0%, rgba(255,255,255,0.02) 100%)`
- **버튼 호버**: `linear-gradient(180deg, rgba(255,255,255,0.1) 0%, rgba(255,255,255,0.05) 100%)`
- **활성 버튼**: `linear-gradient(135deg, #3B82F6 0%, #2563EB 50%, #1D4ED8 100%)`
- **슬라이드 효과**: `linear-gradient(90deg, transparent 0%, rgba(255,255,255,0.1) 50%, transparent 100%)`
- **카테고리 텍스트**: `linear-gradient(135deg, #94A3B8 0%, #60A5FA 50%, #94A3B8 100%)`

### 애니메이션 타이밍
- **Transition**: `0.3s cubic-bezier(0.4, 0, 0.2, 1)` (부드러운 easing)
- **펄스 애니메이션**: `3s ease-in-out infinite`
- **페이드인**: `0.6s ease-out`
- **슬라이드 효과**: `0.5s ease`
- **리플 효과**: `0.6s ease-out`
- **아이콘 회전**: `0.6s ease-in-out` (또는 기울기 효과)
- **그라데이션 텍스트**: `3s ease infinite`

### 그림자 및 글로우
- **호버 그림자**: `0 4px 16px rgba(0, 0, 0, 0.3), 0 0 0 1px rgba(255, 255, 255, 0.1)`
- **활성 글로우**: 펄스 애니메이션으로 변화
  - 최소: `0 4px 12px rgba(59, 130, 246, 0.3), 0 0 0 0 rgba(59, 130, 246, 0.4)`
  - 최대: `0 6px 20px rgba(59, 130, 246, 0.4), 0 0 0 4px rgba(59, 130, 246, 0.1)`

### Transform 효과
- **호버 시**: `scale(1.01) rotate(0.5deg)` (미묘한 확대 및 기울기)
- **페이드인**: `translateY(-4px)` → `translateY(0)` (카테고리 제목만)
- **리플**: `scale(0)` → `scale(4)` (클릭 시)

### 백드롭 블러
- **블러 강도**: `blur(8px)`
- **배경 오버레이**: `rgba(15, 23, 42, 0.6)`
- **브라우저 호환**: `backdrop-filter` + `-webkit-backdrop-filter`

## ⚠️ 주의사항

### 1. Transform 사용 최소화
- `scale(1.01) rotate(0.5deg)`만 사용 (레이아웃 영향 최소)
- `translateX`는 절대 사용 안 함
- 카테고리 제목 페이드인에만 `translateY` 사용 (애니메이션 종료 후 원위치)
- 리플 효과는 `scale`만 사용 (중앙 기준)

### 2. 성능 고려
- 애니메이션은 `transform`과 `opacity` 중심 (GPU 가속)
- 복잡한 애니메이션은 최소화
- `will-change` 속성 사용 고려 (필요시)
- 백드롭 블러는 성능에 영향 가능 (8px로 제한)

### 3. 접근성
- 애니메이션은 시각적 향상용
- 기능에는 영향 없음
- `prefers-reduced-motion` 미디어 쿼리 고려 (선택사항)

### 4. 브라우저 호환성
- 그라데이션 텍스트: `-webkit-background-clip: text` 사용 (fallback 제공)
- 백드롭 블러: `backdrop-filter` + `-webkit-backdrop-filter` 사용
- 리플 효과: 모든 브라우저 지원 (`transform`, `opacity` 사용)

### 5. 아이콘 회전 효과 제한
- CSS만으로 버튼 내부 이모지/아이콘만 선택하는 것은 제한적
- 대안: 버튼 전체에 미묘한 기울기 효과 (`rotate(0.5deg)`)
- 또는 버튼 호버 시 미묘한 전체 회전 효과

## 💡 추가 고급 효과 상세 구현

### 1. 버튼 아이콘 회전 효과

**구현 방법:**
- CSS만으로는 버튼 내부 이모지/아이콘만 선택하기 어려움
- **대안**: 버튼 호버 시 전체에 미묘한 기울기 효과
- `transform: scale(1.01) rotate(0.5deg)` 사용

**CSS:**
```css
.ps-sidebar-scope .stButton > button:hover {
    transform: scale(1.01) rotate(0.5deg);
}
```

### 2. 리플 효과

**구현 방법:**
- `::after` 가상 요소 사용
- `:active` 상태에서 크기 확대 애니메이션
- `premium-ripple` keyframes 사용

**CSS:**
```css
.ps-sidebar-scope .stButton > button::after {
    content: '';
    position: absolute;
    top: 50%;
    left: 50%;
    width: 0;
    height: 0;
    border-radius: 50%;
    background: rgba(255, 255, 255, 0.3);
    transform: translate(-50%, -50%);
    pointer-events: none;
    z-index: 2;
}

.ps-sidebar-scope .stButton > button:active::after {
    width: 300px;
    height: 300px;
    animation: premium-ripple 0.6s ease-out;
}
```

### 3. 그라데이션 텍스트

**구현 방법:**
- `background: linear-gradient()` 사용
- `-webkit-background-clip: text` 및 `-webkit-text-fill-color: transparent` 사용
- `background-clip: text` (표준)
- 애니메이션으로 그라데이션 이동

**CSS:**
```css
.ps-sidebar-scope .premium-category-title {
    background: linear-gradient(135deg, 
        #94A3B8 0%, 
        #60A5FA 50%, 
        #94A3B8 100%);
    background-size: 200% 200%;
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
    background-clip: text;
    animation: premium-gradient-text 3s ease infinite;
    /* Fallback */
    color: #94A3B8;
}
```

### 4. 백드롭 블러

**구현 방법:**
- `.ps-sidebar-scope::before` 사용
- `backdrop-filter: blur(8px)` 적용
- 반투명 배경 오버레이
- 콘텐츠는 `z-index: 1`로 위에 표시

**CSS:**
```css
.ps-sidebar-scope::before {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background: rgba(15, 23, 42, 0.6);
    backdrop-filter: blur(8px);
    -webkit-backdrop-filter: blur(8px);
    pointer-events: none;
    z-index: -1;
}

.ps-sidebar-scope > * {
    position: relative;
    z-index: 1;
}
```
