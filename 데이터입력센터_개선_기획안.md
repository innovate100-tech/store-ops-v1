**우선순위 로직**:
```
1단계: 메뉴 입력 (가장 기본)
2단계: 재료 입력 (메뉴와 연결)
3단계: 레시피 입력 (메뉴+재료 필요)
4단계: 재고 입력 (재료 필요)
5단계: 일일 마감 (운영 시작)
6단계: 목표 설정 (판단 기준)
```

**디자인**:
- 파란색 그라데이션 배지
- "STEP 1", "STEP 2" 형태
- 카드 상단 또는 제목 옆에 표시

**1.1.2 진행률에 구체적 정보 추가**

```python
# 예시: 게이지 바 옆에 텍스트 추가
def _render_progress_with_text(current, total, item_name):
    """
    진행률을 구체적인 숫자로 표시
    
    예: "메뉴 15개 중 12개 입력됨 (3개 남음)"
    """
    remaining = total - current
    progress_text = f"""
    <div style="
        font-size: 0.85rem;
        color: #94A3B8;
        margin-top: 0.5rem;
    ">
        {item_name} <strong style="color: #60A5FA;">{current}개</strong> 중 
        <strong style="color: #10B981;">{current}개</strong> 입력됨
        {f'<span style="color: #F59E0B;">({remaining}개 남음)</span>' if remaining > 0 else '<span style="color: #10B981;">(완료!)</span>'}
    </div>
    """
    return progress_text
```

**표시 위치**:
- 게이지 바 바로 아래
- 작은 회색 텍스트로 표시
- 남은 개수는 주황색으로 강조

```python
def render_detailed_progress(current, total, item_name, completed_items=None):
    """
    상세한 진행 상황 표시
    
    예: 
    - "메뉴 15개 중 12개 입력됨 (3개 남음)"
    - 완료된 항목 목록 표시
    """
    remaining = total - current
    percentage = (current / total * 100) if total > 0 else 0
    
    progress_html = f"""
    <div style="margin-top: 0.5rem;">
        <div style="
            display: flex;
            justify-content: space-between;
            align-items: center;
            margin-bottom: 0.5rem;

            </span>
            <span style="
                font-size: 0.85rem;
                font-weight: 600;
                color: {'#10B981' if percentage >= 80 else '#F59E0B' if percentage >= 50 else '#EF4444'};
            ">
                {percentage:.0f}%
            </span>
        </div>
        
        {f'<div style="font-size: 0.8rem; color: #F59E0B; margin-top: 0.25rem;">⚠️ {remaining}개 남음</div>' if remaining > 0 else '<div style="font-size: 0.8rem; color: #10B981; margin-top: 0.25rem;">✅ 완료!</div>'}
        
        {f'''
        <div style="
            margin-top: 0.5rem;
            font-size: 0.75rem;
            color: #64748B;
        ">
            완료된 항목: {', '.join(completed_items[:5]) if completed_items else '없음'}
            {f' 외 {len(completed_items) - 5}개' if completed_items and len(completed_items) > 5 else ''}
        </div>
        ''' if completed_items else ''}
    </div>
    """
    return progress_html
```
