# 데이터 입력 센터 본문 제목 변경 기획안

## 1. 문제 상황
데이터 입력 센터 페이지의 본문에 표시되는 제목들이 여전히 변경되지 않은 상태로 보입니다.

## 2. 변경 대상 제목 목록

사용자가 확인한 변경되지 않은 제목들:
1. ✍ 입력 허브
2. 📊 실시간 입력 현황
3. 🏗️ 가게 데이터 기초 체력
4. ⚡ 매일 · 매주 · 매월 루틴
5. 🎯 목표 및 분석 기준
6. 🛠️ 가게 정의 (기초 뼈대)
7. ⚙️ 데이터 보정 도구 (과거 일괄 수정)

## 3. 변경 목표

모든 제목을 입력 중심으로 통일:

| 현재 제목 | 변경 후 제목 | 코드 위치 | 변경 방법 |
|---------|------------|---------|---------|
| ✍ 입력 허브 | ✍ 데이터 입력 센터 | 139줄 `render_page_header()` | 함수 호출 인자 변경 |
| 📊 실시간 입력 현황 | 📊 오늘의 입력 현황 | 205줄 `st.markdown("### ...")` | 마크다운 텍스트 변경 |
| 🏗️ 가게 데이터 기초 체력 | 🏗️ 입력 데이터 완성도 | 219줄 `st.markdown("### ...")` | 마크다운 텍스트 변경 |
| ⚡ 매일 · 매주 · 매월 루틴 | ⚡ 정기 입력 작업 | 243줄 `st.markdown("#### ...")` | 마크다운 텍스트 변경 |
| 🎯 목표 및 분석 기준 | 🎯 목표 입력 | 258줄 `st.markdown("#### ...")` | 마크다운 텍스트 변경 |
| 🛠️ 가게 정의 (기초 뼈대) | 🛠️ 기초 데이터 입력 | 271줄 `st.markdown("#### ...")` | 마크다운 텍스트 변경 |
| ⚙️ 데이터 보정 도구 (과거 일괄 수정) | ⚙️ 과거 데이터 입력 | 293줄 `st.expander()` | Expander 제목 변경 |

## 4. 상세 변경 내역

### 4.1 페이지 헤더 (139줄)
```python
# 변경 전
render_page_header("✍ 입력 허브", "✍")

# 변경 후
render_page_header("✍ 데이터 입력 센터", "✍")
```

### 4.2 섹션 제목 1: 실시간 입력 현황 (205줄)
```python
# 변경 전
st.markdown("### 📊 실시간 입력 현황")

# 변경 후
st.markdown("### 📊 오늘의 입력 현황")
```

### 4.3 섹션 제목 2: 가게 데이터 기초 체력 (219줄)
```python
# 변경 전
st.markdown("### 🏗️ 가게 데이터 기초 체력")

# 변경 후
st.markdown("### 🏗️ 입력 데이터 완성도")
```

### 4.4 섹션 제목 3: 매일 · 매주 · 매월 루틴 (243줄)
```python
# 변경 전

# 변경 후
st.markdown("#### ⚡ 정기 입력 작업")
```

### 4.5 섹션 제목 4: 목표 및 분석 기준 (258줄)
```python

```

### 4.6 섹션 제목 5: 가게 정의 (271줄)
```python

# 변경 후
st.markdown("#### 🛠️ 기초 데이터 입력")
```

### 4.7 Expander 제목: 데이터 보정 도구 (293줄)
```python
# 변경 전

# 변경 후

```

## 7. 주의사항

### 7.1 정확한 문자열 매칭
- 마크다운 제목의 정확한 문자열을 찾아 변경해야 함
- 공백, 특수문자, 이모지까지 정확히 일치해야 함

### 7.2 Expander 제목
- `st.expander()`의 첫 번째 인자가 제목이므로 정확히 변경 필요

### 7.3 파일 저장 및 재시작
- 변경 후 파일 저장 필수
- Streamlit 앱 재시작 필요
- 브라우저 캐시 클리어 권장

## 8. 문제 해결 가이드

### 8.1 변경이 반영되지 않는 경우
1. **파일 저장 확인**: 파일이 실제로 저장되었는지 확인
2. **앱 재시작**: Streamlit 앱을 완전히 종료 후 재시작
3. **브라우저 캐시**: Ctrl+F5로 강력 새로고침
4. **파일 경로 확인**: 올바른 파일을 수정했는지 확인
