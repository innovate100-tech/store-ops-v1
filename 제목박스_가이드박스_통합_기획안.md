# 제목 박스 + 가이드 박스 통합 기획안
## 공간 절약하면서도 멋있게! 🎨

---

## 📋 현재 상황 분석

### 현재 구조
```
┌─────────────────────────────────┐
│  [제목 박스]                    │
│  - 아이콘 + "데이터 입력 센터"  │
│  - 부제목                        │
│  - 파란색 네온 바                │
└─────────────────────────────────┘
         ↓ (여백)
┌─────────────────────────────────┐
│  [가이드 박스]                  │
│  - "데이터 자산 가이드" 제목     │
│  - 설명 텍스트                   │
│  - MATURITY LEVEL 75%           │
│  - 진행률 바                     │
│  - 불릿 포인트 2개               │
└─────────────────────────────────┘
```

**문제점**:
- 두 박스가 세로로 배치되어 **수직 공간을 많이 차지**
- 중간 여백으로 인한 공간 낭비
- 스크롤이 길어져 사용성이 저하

---

## 🎯 통합 목표

1. **공간 절약**: 수직 공간 40-50% 절감
2. **디자인 유지**: 현재의 프리미엄 느낌 보존
3. **정보 보존**: 모든 정보를 빠짐없이 표시
4. **가독성 유지**: 정보가 압축되어도 읽기 쉬워야 함

---

## 💡 통합 옵션 제안

### 옵션 1: 좌우 분할 레이아웃 (추천 ⭐⭐⭐⭐⭐)

**개념**: 제목은 왼쪽, 가이드는 오른쪽으로 배치

```
┌─────────────────────────────────────────────────────┐
│  ┌──────────────┐  ┌─────────────────────────────┐  │
│  │ [제목 영역]  │  │ [가이드 영역]              │  │
│  │              │  │                             │  │
│  │ 📝 아이콘    │  │ 💡 데이터 자산 가이드      │  │
│  │ 데이터 입력  │  │ 설명 텍스트...              │  │
│  │ 센터         │  │                             │  │
│  │              │  │ MATURITY LEVEL: 75%         │  │
│  │ 매장 운영 OS │  │ [=====      ]               │  │
│  │ 조종석       │  │                             │  │
│  │              │  │ • 불릿 포인트 1             │  │
│  │ ────────────│  │ • 불릿 포인트 2             │  │
│  └──────────────┘  └─────────────────────────────┘  │
│  [파란색 네온 바 - 전체 너비]                        │
└─────────────────────────────────────────────────────┘
```

**장점**:
- ✅ 공간 절약 효과 최대 (약 50% 절감)
- ✅ 두 정보를 동시에 볼 수 있어 효율적
- ✅ 시각적 균형이 좋음
- ✅ 반응형으로 모바일에서는 세로 배치 가능

**단점**:
- ⚠️ 가이드 텍스트가 길면 오른쪽이 좁아질 수 있음
- ⚠️ 제목과 가이드의 시각적 계층이 약간 약해질 수 있음

**디자인 특징**:
- 왼쪽: 제목 영역 (40% 너비)
- 오른쪽: 가이드 영역 (60% 너비)
- 하단: 파란색 네온 바 (전체 너비)
- 배경: 동일한 다크 블루 그라데이션
- 테두리: 상단 파란색 네온 바 유지

---

### 옵션 2: 상하 압축형 (추천 ⭐⭐⭐⭐)

**개념**: 제목과 가이드를 하나의 박스에 세로로 압축 배치

```
┌─────────────────────────────────────────────────────┐
│  [파란색 네온 바 - 전체 너비]                        │
│  ┌───────────────────────────────────────────────┐  │
│  │ 📝 데이터 입력 센터                           │  │
│  │ 매장 운영 OS 조종석                            │  │
│  │ ───────────────────────────────────────────── │  │
│  │ 💡 데이터 자산 가이드                         │  │
│  │ 설명 텍스트...                                 │  │
│  │                                                │  │
│  │ MATURITY LEVEL: 75%  [=====      ]            │  │
│  │                                                │  │
│  │ • 불릿 포인트 1                                │  │
│  │ • 불릿 포인트 2                                │  │
│  └───────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

**장점**:
- ✅ 하나의 통합된 박스로 깔끔함
- ✅ 제목의 시각적 우선순위 유지
- ✅ 구현이 간단함

**단점**:
- ⚠️ 공간 절약 효과가 옵션 1보다 작음 (약 30% 절감)
- ⚠️ 여전히 세로로 길어질 수 있음

**디자인 특징**:
- 상단: 파란색 네온 바
- 제목 영역: 패딩 축소 (2.5rem → 1.5rem)
- 구분선: 얇은 회색 선으로 제목과 가이드 구분
- 가이드 영역: 패딩 축소 (2rem → 1.5rem)
- 진행률 바: 한 줄에 MATURITY LEVEL과 바 함께 표시

### 옵션 3: 탭/토글 방식 (추천 ⭐⭐⭐)

**개념**: 제목은 항상 보이고, 가이드는 토글로 접었다 펼 수 있게

```
┌─────────────────────────────────────────────────────┐
│  [파란색 네온 바]                                    │
│  ┌───────────────────────────────────────────────┐  │
│  │ 📝 데이터 입력 센터                           │  │
│  │ 매장 운영 OS 조종석                            │  │
│  │                                                │  │
│  │ [▼ 데이터 자산 가이드 보기] (토글 버튼)      │  │
│  └───────────────────────────────────────────────┘  │
│                                                      │
│  (토글 클릭 시 아래 펼쳐짐)                          │
│  ┌───────────────────────────────────────────────┐  │
│  │ 💡 데이터 자산 가이드                         │  │
│  │ 설명 텍스트...                                 │  │
│  │ MATURITY LEVEL: 75%  [=====      ]            │  │
│  │ • 불릿 포인트 1                                │  │
│  │ • 불릿 포인트 2                                │  │
│  └───────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

**장점**:
- ✅ 공간 절약 효과 최대 (기본 상태에서 거의 0)
- ✅ 사용자가 필요할 때만 가이드 확인 가능
- ✅ 깔끔한 초기 화면

**단점**:
- ⚠️ 가이드 정보가 숨겨져 있어 첫 방문자가 놓칠 수 있음
- ⚠️ 토글 인터랙션이 추가되어 복잡도 증가

**디자인 특징**:
- 기본 상태: 제목만 표시 (최소 공간)
- 토글 버튼: 부드러운 애니메이션으로 펼침/접힘
- 펼쳐진 상태: 가이드 정보 표시

### 옵션 4: 인라인 통합형 (추천 ⭐⭐⭐⭐)

**개념**: 제목과 가이드를 자연스럽게 하나의 흐름으로 통합

```
┌─────────────────────────────────────────────────────┐
│  [파란색 네온 바]                                    │
│  ┌───────────────────────────────────────────────┐  │
│  │ 📝 데이터 입력 센터                           │  │
│  │ 매장 운영 OS 조종석                            │  │
│  │                                                │  │
│  │ 이 앱은 '감'이 아니라 데이터 자산으로 매장을  │  │
│  │ 운영하게 만듭니다. 아래 항목들이 채워질수록,  │  │
│  │ 매장 운영이 시스템이 됩니다.                  │  │
│  │                                                │  │
│  │ MATURITY LEVEL: 75%  [=====      ]            │  │
│  │                                                │  │
│  │ • 비어 있는 데이터를 채우면 정밀 리포트/전략  │  │
│  │   기능이 단계적으로 열립니다.                 │  │
│  │ • 입력은 일이 아니라, 매장의 운영 시스템을    │  │
│  │   만드는 과정입니다.                           │  │
│  │                                                │  │
│  │ 아래 패널들은 현재 매장이 보유한 '데이터 자산' │  │
│  │ 상태입니다.                                    │  │
│  └───────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

**장점**:
- ✅ 하나의 자연스러운 흐름으로 읽기 쉬움
- ✅ "데이터 자산 가이드" 제목 제거로 공간 절약
- ✅ 제목과 설명이 논리적으로 연결됨

**단점**:
- ⚠️ 가이드의 정체성이 약해질 수 있음
- ⚠️ 정보 계층이 약간 모호해질 수 있음

**디자인 특징**:
- 제목: 큰 폰트, 굵게
- 설명: 제목 아래 자연스럽게 이어짐
- 진행률: 중간에 배치
- 불릿 포인트: 설명 아래 배치
- 마지막 문장: 작은 회색 텍스트로 마무리

## 💡 추가 아이디어
