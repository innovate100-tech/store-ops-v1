## 건강검진 카테고리 상세 역할 정의

### [1] 건강검진 실시
- QSCPPPMHF 체크리스트 입력 페이지
- 정기 건강검진 전용
- session 기반
- 완료 시 results/sessions 생성

### [2] 검진 결과 요약
- 가장 최근 완료 검진 기준
- 포함 요소:
  - 9축 점수(Q,S,C,P1,P2,P3,M,H,F)
  - 위험/주의/양호 분류
  - Top3 리스크 자동 요약
  - HOME/전략으로 연결되는 CTA 버튼

### [3] 검진 히스토리
- 검진 회차 리스트
- 날짜별 점수 변화
- 축별 트렌드(Q~F)
- "개선됨 / 악화됨 / 정체" 표시
