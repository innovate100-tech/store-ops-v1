## 📋 상세 기능 명세

### Phase 1: 공급업체 관리 (필수)

#### 1.1 공급업체 등록
```
- 공급업체명
- 연락처 (전화번호, 이메일)
- 배송 정보 (배송일, 최소 주문금액, 배송비)
- 비고
```

#### 1.2 재료-공급업체 매핑
```
- 재료별 기본 공급업체 설정
- 재료별 대체 공급업체 등록
- 공급업체별 단가 관리 (가격 변동 이력)
```

#### 1.3 공급업체별 가격 비교
```
- 동일 재료의 공급업체별 가격 비교
- 최저가 공급업체 자동 추천
- 가격 변동 추이 그래프
```

### Phase 2: 발주 이력 관리 (필수)

#### 2.1 발주 생성
```
- 발주 추천 리스트에서 발주 생성
- 발주일 선택
- 공급업체 선택 (재료별)
- 발주 수량 수정 가능
- 발주 메모 추가
```

#### 2.2 발주 상태 관리
```
- 발주 예정 (생성됨)
- 발주 완료 (공급업체에 전달)
- 입고 완료 (재고 반영)
- 발주 취소
```

#### 2.3 발주 이력 조회
```
- 기간별 발주 이력
- 공급업체별 발주 이력
- 재료별 발주 이력
- 발주 상태 필터링
```

### Phase 3: 스마트 발주 최적화 (고급)

#### 3.1 최소 주문량 고려
```
- 공급업체별 최소 주문금액 확인
- 최소 주문량 미달 시 자동 추가 추천
- 배송비 포함 총 비용 계산
```

#### 3.2 통합 발주 최적화
```
- 동일 공급업체 재료 통합 발주
- 배송비 절감 계산
- 발주 금액 최적화 제안
```

#### 3.3 발주 빈도 최적화
```
- 재고 회전율 기반 최적 발주 주기 계산
- 발주 빈도별 비용 비교
- 최적 발주 시기 추천
```

### Phase 4: 알림 및 경고 (고급)

#### 4.1 품절 위험 알림
```
- 현재고 < 안전재고 시 경고
- 예상 소진일 계산 및 알림
- 긴급 발주 필요 재료 강조
```

#### 4.2 발주 시기 알림
```
- 발주 예정일 알림
- 입고 예정일 알림
- 발주 미완료 재료 리마인더
```

#### 4.3 과다재고 경고
```
- 재고 회전율 낮은 재료 경고
- 유통기한 임박 재료 경고
- 과다재고 비용 계산
```

### Phase 5: 발주 분석 대시보드 (고급)

#### 5.1 재고 회전율 분석
```
- 재료별 재고 회전율
- 평균 재고 보유일수
- 회전율 낮은 재료 TOP 10
```

#### 5.2 발주 패턴 분석
```
- 월별 발주 횟수/금액
- 공급업체별 발주 비중
- 재료별 발주 빈도
- 발주 비용 추이 그래프
```

#### 5.3 비용 최적화 인사이트
```
- 배송비 절감 기회
- 가격 변동 영향 분석
- 발주 최적화 제안
```

## 📈 기대 효과

### 비즈니스 임팩트
1. **품절 방지**: 95% → 99% (알림 시스템)
2. **과다재고 감소**: 20% 감소 (최적화 알고리즘)
3. **발주 비용 절감**: 배송비 15% 절감 (통합 발주)
4. **시간 절약**: 발주 관리 시간 50% 절감

### 사용성 개선
1. **공급업체 관리**: 한 곳에서 모든 공급업체 정보 관리
2. **발주 추적**: 발주 상태 실시간 확인
3. **데이터 기반 의사결정**: 발주 패턴 분석으로 최적화

### 차별화 포인트
1. **스마트 최적화**: 단순 추천이 아닌 비용 최적화
2. **통합 관리**: 발주부터 입고까지 전 과정 관리
3. **예측 분석**: 재고 회전율 기반 발주 시기 예측
