# 🔍 Console에서 [DEBUG] 메시지 확인하기 - 왕초보 가이드

## 📋 목적

JavaScript가 "시작 필요" 카드를 찾았는지 확인하기 위해 Console에서 `[DEBUG]` 메시지를 찾는 방법입니다.

---

## 🎯 빠른 시작 (3단계)

### 1단계: Console 열기
- `F12` 키 누르기
- **Console** 탭 클릭

### 2단계: 메시지 찾기
- `Ctrl + F` (또는 `Cmd + F` on Mac) 눌러서 검색창 열기
- 검색창에 `[DEBUG]` 입력

### 3단계: 결과 확인
- `[DEBUG] X개 카드 발견 (텍스트 기반)` 메시지가 보이면 성공!
- 숫자가 0이 아니면 카드를 찾은 것입니다

---

## 📖 상세 가이드

### 방법 1: 검색 기능 사용 (가장 쉬움)

1. **Console 탭 열기**
   ```
   F12 → Console 탭 클릭
   ```

2. **검색창 열기**
   - `Ctrl + F` (Windows/Linux)
   - `Cmd + F` (Mac)
   - 또는 Console 상단의 검색 아이콘 클릭

3. **검색어 입력**
   - 검색창에 `[DEBUG]` 입력
   - Enter 키 누르기

4. **결과 확인**
   - 검색 결과가 하이라이트됩니다
   - `[DEBUG] 3개 카드 발견 (텍스트 기반)` 같은 메시지 찾기

**예시 화면**:
```
Console
├─ ✅ keyframes 생성 완료!
├─ 🔍 카드 요소 찾는 중...
└─ [DEBUG] 3개 카드 발견 (텍스트 기반)  ← 이 메시지!
```

---

### 방법 2: 스크롤해서 찾기

1. **Console 탭 열기**
   - `F12` → Console 탭

2. **스크롤하기**
   - 마우스 휠로 위아래 스크롤
   - 또는 스크롤바 드래그

3. **메시지 찾기**
   - `[DEBUG]` 라는 글자가 있는 줄 찾기
   - 보통 `keyframes 생성 완료!` 메시지 다음에 나타납니다

4. **메시지 읽기**
   - `[DEBUG] X개 카드 발견 (텍스트 기반)`
   - X가 0보다 크면 성공!

---

### 방법 3: 필터 사용

1. **Console 탭 열기**
   - `F12` → Console 탭

2. **필터 아이콘 클릭**
   - Console 상단의 필터 아이콘 찾기
   - 또는 `Ctrl + Shift + F` (Windows/Linux)

3. **필터 입력**
   - 필터 입력창에 `DEBUG` 입력
   - `[DEBUG]` 메시지만 보이도록 필터링

4. **결과 확인**
   - 필터링된 메시지만 표시됩니다


## 💡 문제 해결

### 메시지가 안 보일 때

1. **페이지 새로고침**
   - `F5` 키 누르기
   - JavaScript가 다시 실행됩니다

2. **잠시 기다리기**
   - JavaScript 실행에 시간이 걸릴 수 있습니다
   - 1-2초 정도 기다린 후 다시 확인

3. **Console 맨 아래 확인**
   - Console을 맨 아래로 스크롤
   - 최신 메시지는 맨 아래에 있습니다

4. **Console 지우기**
   - Console 상단의 🚫 아이콘 클릭
   - 또는 `Ctrl + L` (Windows/Linux)
   - 페이지 새로고침 후 다시 확인

### 메시지가 0개일 때

- 카드를 찾지 못한 것입니다
- Elements 탭에서 "지금 시작하세요" 텍스트가 있는 요소를 확인해주세요
- 페이지가 완전히 로드되지 않았을 수 있습니다 (잠시 후 다시 시도)

2. **`[DEBUG]` 메시지가 보이도록 스크롤**

   - Windows: `Windows + Shift + S`
   - Mac: `Cmd + Shift + 4`
   - 또는 Print Screen 키
