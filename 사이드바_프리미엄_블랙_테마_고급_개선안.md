# 사이드바 프리미엄 블랙 테마 고급 개선안

## 📋 목표

**현재 프리미엄 블랙 테마를 더 고급스럽고 세련되게 개선**
- 고급 애니메이션 효과 추가
- 그라데이션 배경 및 호버 효과 강화
- 미묘한 글로우 및 그림자 효과
- 부드러운 전환 애니메이션

---

## ⚠️ 제약사항 유지

- JavaScript, DOM 조작 금지
- `[data-testid="stSidebar"]` 선택자 사용 금지
- 사이드바 위치/크기/토글 CSS 금지
- 메인 콘텐츠에 영향 주는 전역 CSS 금지
- `.ps-sidebar-scope` 하위로만 CSS 적용
- `transform: translateX()` 금지 (레이아웃 흔들림 방지)

---

## 🎨 고급 개선 요소

### 1. 그라데이션 배경
- 버튼 기본 배경: 미묘한 그라데이션 (위에서 아래로 어두워지는 효과)
- 호버 시: 더 밝은 그라데이션으로 변화
- 활성 버튼: 블루 그라데이션 + 미묘한 애니메이션

### 2. 고급 애니메이션
- CSS `@keyframes`를 사용한 부드러운 펄스 효과
- 호버 시 배경색 그라데이션 변화
- 활성 버튼에 미묘한 글로우 펄스
- 카테고리 제목에 미묘한 페이드인 효과

### 3. 그림자 및 글로우 효과
- 호버 시: 더 깊은 그림자 + 미묘한 글로우
- 활성 버튼: 강화된 글로우 + 펄스 애니메이션
- 계층감 있는 그림자 효과

### 4. 색상 및 그라데이션
- 더 풍부한 그라데이션 팔레트
- 미묘한 색상 변화 애니메이션
- 부드러운 전환 효과

## 🔧 구현 방법

### 1. CSS Keyframes 애니메이션

```css
/* 펄스 글로우 애니메이션 (활성 버튼용) */
@keyframes premium-glow-pulse {
    0%, 100% {
        box-shadow: 0 4px 12px rgba(59, 130, 246, 0.3),
                    0 0 0 0 rgba(59, 130, 246, 0.4);
    }
    50% {
        box-shadow: 0 6px 20px rgba(59, 130, 246, 0.4),
                    0 0 0 4px rgba(59, 130, 246, 0.1);
    }
}

/* 그라데이션 배경 애니메이션 (호버용) */
@keyframes premium-gradient-shift {
    0% {
        background-position: 0% 50%;
    }
    50% {
        background-position: 100% 50%;
    }
    100% {
        background-position: 0% 50%;
    }
}

/* 미묘한 펄스 효과 (카테고리 제목용) */
@keyframes premium-fade-in {
    from {
        opacity: 0;
        transform: translateY(-4px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
```

### 2. 고급 버튼 스타일

```css
/* 공통 버튼: 고급 그라데이션 배경 */
.ps-sidebar-scope .stButton > button {
    background: linear-gradient(180deg, 
        rgba(255, 255, 255, 0.05) 0%, 
        rgba(255, 255, 255, 0.02) 100%);
    color: #E2E8F0;
    border: 1px solid rgba(255, 255, 255, 0.12);
    border-radius: 12px;
    padding: 0.875rem 1rem;
    font-weight: 500;
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
    margin-bottom: 0.5rem;
    text-align: left;
    font-size: 0.875rem;
    position: relative;
    overflow: hidden;
}

/* 버튼 내부 그라데이션 오버레이 (호버 효과용) */
.ps-sidebar-scope .stButton > button::before {
    content: '';
    position: absolute;
    top: 0;
    left: -100%;
    width: 100%;
    height: 100%;
    background: linear-gradient(90deg, 
        transparent 0%, 
        rgba(255, 255, 255, 0.1) 50%, 
        transparent 100%);
    transition: left 0.5s ease;
}

/* 호버 시: 그라데이션 배경 변화 + 슬라이드 효과 */
.ps-sidebar-scope .stButton > button:hover {
    background: linear-gradient(180deg, 
        rgba(255, 255, 255, 0.1) 0%, 
        rgba(255, 255, 255, 0.05) 100%);
    border-color: rgba(255, 255, 255, 0.25);
    box-shadow: 0 4px 16px rgba(0, 0, 0, 0.3),
                0 0 0 1px rgba(255, 255, 255, 0.1);
    transform: scale(1.01);
}

.ps-sidebar-scope .stButton > button:hover::before {
    left: 100%;
}

/* 활성 버튼: 고급 블루 그라데이션 + 펄스 애니메이션 */
.ps-sidebar-scope .stButton > button[kind="primary"] {
    background: linear-gradient(135deg, 
        #3B82F6 0%, 
        #2563EB 50%, 
        #1D4ED8 100%);
    background-size: 200% 200%;
    border-color: #60A5FA;
    box-shadow: 0 4px 16px rgba(59, 130, 246, 0.4),
                0 0 0 1px rgba(96, 165, 250, 0.3);
    color: #FFFFFF;
    animation: premium-glow-pulse 3s ease-in-out infinite;
    position: relative;
}

/* 활성 버튼 내부 미묘한 빛 효과 */
.ps-sidebar-scope .stButton > button[kind="primary"]::after {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    height: 50%;
    background: linear-gradient(180deg, 
        rgba(255, 255, 255, 0.2) 0%, 
        transparent 100%);
    border-radius: 12px 12px 0 0;
    pointer-events: none;
}
```

## 🎨 디자인 상세 스펙

### 색상 및 그라데이션
- **버튼 기본**: `linear-gradient(180deg, rgba(255,255,255,0.05) 0%, rgba(255,255,255,0.02) 100%)`
- **버튼 호버**: `linear-gradient(180deg, rgba(255,255,255,0.1) 0%, rgba(255,255,255,0.05) 100%)`
- **활성 버튼**: `linear-gradient(135deg, #3B82F6 0%, #2563EB 50%, #1D4ED8 100%)`
- **슬라이드 효과**: `linear-gradient(90deg, transparent 0%, rgba(255,255,255,0.1) 50%, transparent 100%)`

### 애니메이션 타이밍
- **Transition**: `0.3s cubic-bezier(0.4, 0, 0.2, 1)` (부드러운 easing)
- **펄스 애니메이션**: `3s ease-in-out infinite`
- **페이드인**: `0.6s ease-out`
- **슬라이드 효과**: `0.5s ease`

### 그림자 및 글로우
- **호버 그림자**: `0 4px 16px rgba(0, 0, 0, 0.3), 0 0 0 1px rgba(255, 255, 255, 0.1)`
- **활성 글로우**: 펄스 애니메이션으로 변화
  - 최소: `0 4px 12px rgba(59, 130, 246, 0.3), 0 0 0 0 rgba(59, 130, 246, 0.4)`
  - 최대: `0 6px 20px rgba(59, 130, 246, 0.4), 0 0 0 4px rgba(59, 130, 246, 0.1)`

### Transform 효과
- **호버 시**: `scale(1.01)` (미묘한 확대, 레이아웃 영향 최소)
- **페이드인**: `translateY(-4px)` → `translateY(0)` (카테고리 제목만)

## ⚠️ 주의사항

### 1. Transform 사용 최소화
- `scale(1.01)`만 사용 (레이아웃 영향 최소)
- `translateX`는 절대 사용 안 함
- 카테고리 제목 페이드인에만 `translateY` 사용 (애니메이션 종료 후 원위치)

### 2. 성능 고려
- 애니메이션은 `transform`과 `opacity` 중심 (GPU 가속)
- 복잡한 애니메이션은 최소화
- `will-change` 속성 사용 고려 (필요시)

### 3. 접근성
- 애니메이션은 시각적 향상용
- 기능에는 영향 없음
- `prefers-reduced-motion` 미디어 쿼리 고려 (선택사항)
