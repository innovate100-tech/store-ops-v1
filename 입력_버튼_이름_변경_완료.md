## 📝 변경 사항 상세

### 1. 일일 마감 → 일일 마감 입력
- **위치**: 입력 허브 빠른 입력 섹션, 스마트 추천
- **변경 이유**: "입력"을 명시하여 페이지 목적 명확화

### 2. 매장 체크리스트 → QSC 입력
- **위치**: 입력 허브 정기 입력 섹션, 스마트 추천
- **변경 이유**: QSC(Quality, Service, Cleanliness) 용어 사용으로 간결화

### 3. 월간 정산 → 월간 정산 입력
- **위치**: 입력 허브 정기 입력 섹션, 스마트 추천
- **변경 이유**: "입력"을 명시하여 페이지 목적 명확화

### 4. 메뉴 입력 → 판매 메뉴 입력
- **위치**: 입력 허브 매장 자산 입력 섹션, 사이드바
- **변경 이유**: "판매"를 명시하여 메뉴의 성격 명확화

### 5. 재료 입력 → 사용 재료 입력
- **위치**: 입력 허브 매장 자산 입력 섹션, 사이드바
- **변경 이유**: "사용"을 명시하여 재료의 용도 명확화

### 6. 레시피 입력 → 판매 레시피 입력
- **위치**: 입력 허브 매장 자산 입력 섹션, 사이드바
- **변경 이유**: "판매"를 명시하여 레시피의 성격 명확화

### 7. 매출 등록(과거/수정) → 일괄 매출/방문자 등록
- **위치**: 입력 허브 보정·특수 입력 섹션, 사이드바
- **변경 이유**: "일괄"로 기능 명확화, "방문자" 포함으로 범위 명시

### 8. 판매량 등록(과거/수정) → 일괄 메뉴별 판매량 등록
- **위치**: 입력 허브 보정·특수 입력 섹션, 사이드바
- **변경 이유**: "일괄", "메뉴별"로 기능과 범위 명확화
