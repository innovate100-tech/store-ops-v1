# 🚀 "시작 필요" 상태 강조 개선 기획서

## 📋 개요

**목적**: "시작 필요" 상태일 때 사용자가 즉시 액션을 취하도록 시각적·기능적 강조 강화

**배경**: 현재 "시작 필요" 상태는 회색(#94A3B8)으로 표시되어 있어 시각적 중요도가 낮아 보임. 첫 시작이 가장 중요한 단계이므로 더 강력한 유도가 필요함.

**기대 효과**: 
- 첫 시작 전환율 향상
- 사용자 혼란 감소 (무엇을 해야 할지 명확히)
- 데이터 자산 구축 속도 향상

---

## 🎯 개선 목표

1. **시각적 강조**: "시작 필요" 상태를 한눈에 알아볼 수 있게
2. **액션 유도**: 즉시 시작할 수 있도록 명확한 경로 제시
3. **심리적 동기부여**: "지금 시작하면 이렇게 좋아진다" 메시지 전달

---

## 💡 개선 아이디어

### 1. 시각적 강조 강화

#### 1.1 색상 변경
- **현재**: 회색 (#94A3B8) - 비활성 느낌
- **개선**: 
  - 주황색 (#F59E0B) 또는 빨간색 계열 (#EF4444) 사용
  - 또는 펄스 애니메이션이 있는 주황색 (#F59E0B) + 글로우 효과

#### 1.2 애니메이션 효과
- **펄스 애니메이션**: 카드가 부드럽게 깜빡이는 효과
- **글로우 효과**: 테두리에 네온 글로우 (주황색 계열)
- **점멸 아이콘**: ⚠️ 또는 🔴 아이콘 추가

#### 1.3 배지 추가
- "시작 필요" 텍스트 옆에 **"NEW"** 또는 **"필수"** 배지 추가
- 또는 **"지금 시작"** 배지

---

### 2. 메시지 강화

#### 2.1 상태 텍스트 변경
- **현재**: "시작 필요"
- **개선 옵션**:
  - "🚨 지금 시작하세요"
  - "⚡ 빠른 시작 필요"
  - "🎯 첫 단계 시작"
  - "✨ 지금 시작하면 분석 가능"

#### 2.2 설명 텍스트 추가
- 상태 카드 하단에 짧은 설명 추가:
  - 예: "메뉴를 등록하면 수익 분석이 시작됩니다"
  - 예: "재료 정보가 있어야 원가 계산이 가능합니다"

---

### 3. 게이지 영역 강조

#### 3.1 진행률 0%일 때 특별 처리
- **현재**: 회색 게이지 바
- **개선**:
  - 빨간색/주황색 게이지 바
  - "0%" 대신 "시작 필요" 텍스트
  - 또는 "0% → 시작하세요" 형태

#### 3.2 경고 메시지 추가
- 게이지 카드 상단에 경고 배너 추가:
  ```
  ⚠️ 이 항목이 없으면 [기능명]을 사용할 수 없습니다
  ```

---

### 4. 버튼 강조

#### 4.1 버튼 스타일 변경
- **"시작 필요" 상태일 때**:
  - 버튼 텍스트: "보완" → "🚀 지금 시작"
  - 버튼 색상: 더 밝은 주황색/빨간색 계열
  - 글로우 효과 추가
  - 펄스 애니메이션

#### 4.2 우선순위 표시
- 버튼 옆에 "1순위" 또는 "필수" 배지 추가

---

### 5. 알림/경고 영역 추가

#### 5.1 패널 상단 경고 배너
- "시작 필요" 항목이 있을 때 패널 상단에 경고 배너 표시:
  ```
  ⚠️ [항목명]이 없어서 [기능명]을 사용할 수 없습니다
  → [버튼] 지금 시작하기
  ```

#### 5.2 첫 시작 가이드
- "시작 필요" 상태일 때 클릭하면 간단한 가이드 툴팁 표시
- 또는 expander로 "첫 시작 가이드" 제공

---

### 6. 우선순위 표시

#### 6.1 시작 순서 가이드
- 여러 항목이 "시작 필요"일 때 우선순위 표시:
  ```
  1️⃣ 메뉴 (먼저 시작)
  2️⃣ 재료 (다음 시작)
  3️⃣ 레시피 (나중에 시작)
  ```

#### 6.2 의존성 표시
- 예: "레시피는 메뉴와 재료가 있어야 시작할 수 있습니다"
