4. Network 탭에서 CSS 로드 확인하기

## 1️⃣ 개발자 도구 열기

### 방법 1: 키보드 단축키 (가장 빠름)
- **Windows/Linux**: `F12` 키 누르기
- **Mac**: `Cmd + Option + I` (또는 `F12`)

### 방법 2: 마우스로 열기
1. 페이지에서 **마우스 오른쪽 버튼** 클릭
2. 메뉴에서 **"검사"** 또는 **"Inspect"** 클릭

### 방법 3: 브라우저 메뉴에서
- **Chrome/Edge**: 
  - 오른쪽 위 점 3개(⋮) 클릭 → **"도구 더보기"** → **"개발자 도구"**
- **Firefox**: 
  - 오른쪽 위 점 3개(⋮) 클릭 → **"웹 개발자"** → **"웹 콘솔"**

## 2️⃣ Elements 탭에서 애니메이션 확인하기

### 단계 1: Elements 탭 열기
1. 개발자 도구가 열리면 상단에 탭들이 보입니다
2. **"Elements"** (또는 **"요소"**) 탭 클릭

### 단계 2: 카드 요소 찾기
1. Elements 탭 왼쪽에 HTML 코드가 보입니다
2. `Ctrl + F` (또는 `Cmd + F` on Mac) 눌러서 검색창 열기
3. 검색창에 `ps-start-needed-card` 입력
4. 검색 결과로 나온 요소를 클릭

### 단계 3: 스타일 확인하기
1. 오른쪽 패널에서 **"Styles"** (또는 **"스타일"**) 탭 확인
2. 아래 항목들을 확인:

#### ✅ 애니메이션이 적용되어 있다면:
```
.ps-start-needed-card {
    animation: pulse-start-needed 2s ease-in-out infinite,
               glow-pulse 3s ease-in-out infinite;
    animation-name: pulse-start-needed, glow-pulse;
    animation-duration: 2s, 3s;
    ...
}
```
또는 **"element.style"** 섹션에:
```
animation: pulse-start-needed 2s ease-in-out infinite, glow-pulse 3s ease-in-out infinite;
```
이런 내용이 보이면 애니메이션이 적용된 것입니다!

#### ❌ 애니메이션이 없다면:
- `animation` 속성이 보이지 않거나
- `animation: none` 이라고 되어 있거나
- 아무것도 없으면 애니메이션이 적용되지 않은 것입니다

**💡 팁**: Styles 탭에서 `Ctrl + F` (또는 `Cmd + F`)를 눌러서 `animation`을 검색하면 빠르게 찾을 수 있습니다!

### 단계 4: Computed 스타일 확인하기
1. 오른쪽 패널에서 **"Computed"** (또는 **"계산된 스타일"**) 탭 클릭
2. 검색창에 `animation` 입력
3. 값이 `none`이면 애니메이션이 적용되지 않은 것입니다

## 3️⃣ Console 탭에서 오류 확인하기

### 단계 1: Console 탭 열기
1. 개발자 도구 상단에서 **"Console"** (또는 **"콘솔"**) 탭 클릭

### 단계 2: 오류 확인하기
1. 빨간색으로 표시된 메시지가 있는지 확인
2. 일반적인 오류 예시:
   - `Uncaught TypeError: ...` → JavaScript 오류
   - `Failed to load resource: ...` → 파일 로드 실패
   - `ReferenceError: ...` → 변수/함수 찾을 수 없음

### 단계 2-1: Console 메시지 종류 이해하기

Console에는 여러 종류의 메시지가 있습니다:

- **✅ 초록색 체크**: 성공 메시지
- **❌ 빨간색 X**: 오류 메시지
- **⚠️ 노란색 경고**: 경고 메시지
- **💡 노란색 전구**: 정보/팁 메시지
- **🔍 파란색 돋보기**: 검색/찾기 메시지
- **📊 파란색 차트**: 데이터/통계 메시지
- **일반 텍스트**: 일반 로그 메시지

**`[DEBUG]` 메시지는 일반 텍스트로 표시됩니다!**

### 단계 3: 애니메이션 관련 로그 확인
1. Console에서 `ps-start-needed` 검색
2. 관련 로그가 있는지 확인

### 단계 3-1: [DEBUG] 메시지 확인하기 (왕초보 버전)

**목적**: JavaScript가 카드를 찾았는지 확인하기

**단계별 방법**:

1. **Console 탭 열기**
   - `F12` 키 누르기
   - **Console** 탭 클릭

2. **메시지 찾기**
   - Console 화면을 위아래로 스크롤
   - `[DEBUG]` 라는 글자가 있는 줄 찾기
   - 또는 `Ctrl + F` (또는 `Cmd + F` on Mac) 눌러서 검색창 열기
   - 검색창에 `[DEBUG]` 입력

3. **메시지 확인**
   - `[DEBUG] X개 카드 발견 (텍스트 기반)` 같은 메시지가 보이면 성공!
   - 예시:
     ```
     [DEBUG] 3개 카드 발견 (텍스트 기반)
     ```
   - 숫자가 0이면 카드를 찾지 못한 것입니다

4. **메시지가 안 보이면**
   - 페이지를 새로고침 (`F5`)
   - 잠시 기다리기 (JavaScript가 실행되는 시간 필요)
   - Console을 맨 아래로 스크롤해서 최신 메시지 확인

**💡 팁**: 
- Console 메시지는 시간순으로 쌓입니다
- 가장 최근 메시지는 맨 아래에 있습니다
- `[DEBUG]` 메시지는 페이지가 로드될 때마다 나타납니다

### 단계 4: 수동으로 테스트하기

⚠️ **중요**: Console에 코드를 붙여넣을 때는 **마크다운 코드 블록(```javascript)을 제외**하고 순수 JavaScript 코드만 붙여넣으세요!

Console 하단 입력창에 다음 코드를 **한 줄씩** 입력하고 Enter:

**1단계: 카드 찾기**
```javascript
const cards = document.querySelectorAll('.ps-start-needed-card');
console.log('찾은 카드 개수:', cards.length);
```

**2단계: 애니메이션 확인**
```javascript
if (cards.length > 0) {
    const style = window.getComputedStyle(cards[0]);
    console.log('애니메이션:', style.animation);
    console.log('애니메이션 이름:', style.animationName);
}
```

**3단계: 애니메이션 강제 적용 (애니메이션이 없을 때)**
```javascript
// keyframes 먼저 생성
const style = document.createElement('style');
style.id = 'ps-animation-fix';
style.textContent = `
    @keyframes pulse-start-needed {
        0%, 100% { transform: scale(1); box-shadow: 0 0 15px rgba(245, 158, 11, 0.5); }
        50% { transform: scale(1.02); box-shadow: 0 0 25px rgba(245, 158, 11, 0.8); }
    }
    @keyframes glow-pulse {
        0%, 100% { box-shadow: 0 0 10px rgba(245, 158, 11, 0.5); }
        50% { box-shadow: 0 0 20px rgba(245, 158, 11, 0.7); }
    }
`;
document.head.appendChild(style);

// 카드에 애니메이션 강제 적용
const cards = document.querySelectorAll('.ps-start-needed-card');
cards.forEach(card => {
    card.style.animation = 'pulse-start-needed 2s ease-in-out infinite, glow-pulse 3s ease-in-out infinite';
    card.style.setProperty('animation', 'pulse-start-needed 2s ease-in-out infinite, glow-pulse 3s ease-in-out infinite', 'important');
    console.log('애니메이션 적용 완료!');
});
```

결과가 콘솔에 출력되고, 카드가 움직이기 시작하면 성공입니다!

## 4️⃣ Network 탭에서 CSS 로드 확인하기

### 단계 1: Network 탭 열기
1. 개발자 도구 상단에서 **"Network"** (또는 **"네트워크"**) 탭 클릭

### 단계 2: 페이지 새로고침
1. `F5` 키 누르기 (또는 새로고침 버튼 클릭)
2. Network 탭에서 파일들이 로드되는 것을 볼 수 있습니다

### 단계 3: CSS 파일 필터링
1. Network 탭 상단에 필터 버튼들이 있습니다
2. **"CSS"** 버튼 클릭 (또는 `Ctrl + Shift + E`)

### 단계 4: CSS 로드 상태 확인
1. CSS 파일 목록이 보입니다
2. 각 파일의 상태 확인:
   - **200** (또는 초록색) = 성공적으로 로드됨 ✅
   - **404** (또는 빨간색) = 파일을 찾을 수 없음 ❌
   - **304** = 캐시에서 로드됨 (정상) ✅

### 단계 5: 스타일시트 내용 확인
1. CSS 파일을 클릭
2. **"Response"** (또는 **"응답"**) 탭 클릭
3. `pulse-start-needed` 또는 `glow-pulse` 검색
4. keyframes 정의가 있는지 확인

## 5️⃣ 실시간으로 애니메이션 확인하기

### 방법 1: Elements에서 직접 확인
1. Elements 탭에서 `.ps-start-needed-card` 요소 선택
2. 오른쪽 패널에서 **"Styles"** 탭
3. `animation` 속성에 마우스 올리기
4. 애니메이션이 작동 중이면 요소가 살짝 움직이거나 글로우 효과가 보입니다

### 방법 2: Console에서 강제 적용 테스트
Console에 다음 코드 입력:

```javascript
// 애니메이션 강제 적용 테스트
const testCard = document.querySelector('.ps-start-needed-card');
if (testCard) {
    testCard.style.animation = 'pulse-start-needed 2s ease-in-out infinite';
    console.log('애니메이션 적용 완료!');
} else {
    console.log('카드를 찾을 수 없습니다.');
}
```

카드가 움직이기 시작하면 애니메이션이 작동하는 것입니다!

## 🆘 문제 해결

### 개발자 도구가 안 열릴 때
- 브라우저를 완전히 종료 후 다시 시작
- 다른 브라우저 시도 (Chrome, Edge, Firefox)

### Elements 탭이 안 보일 때
- 개발자 도구 상단 탭 메뉴 확인
- `>>` 아이콘 클릭하여 숨겨진 탭 보기

### Console에 아무것도 안 보일 때
- Console 상단 필터 아이콘 확인
- "All levels" 선택되어 있는지 확인
