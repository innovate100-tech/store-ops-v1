# 데이터 입력센터 제목 디자인 개선 기획서

## 📋 개요

**목적**: 데이터 입력센터 페이지의 제목을 현재 페이지의 프리미엄 다크 테마와 조화롭게 꾸며 시각적 임팩트를 강화

**현재 상태**: 
- `render_page_header("✍ 데이터 입력 센터", "✍")` 사용
- 기본적인 아이콘 + 제목 + 파란색 그라데이션 바 형태
- 페이지의 프리미엄 스타일과 비교 시 제목이 다소 단조로움

**목표**: 
- 페이지의 "Control Board" 컨셉과 조화
- 프리미엄 다크 테마와 일관성 유지
- 시각적 계층 구조 강화

---

## 🎨 현재 페이지 스타일 분석

### 색상 팔레트
- **배경**: `rgba(15, 23, 42, 0.95)` ~ `rgba(30, 41, 59, 0.9)` (다크 슬레이트)
- **주요 강조색**: `#3B82F6` (파란색), `#60A5FA` (밝은 파란색)
- **텍스트**: `#F8FAFC` (밝은 회색), `#E2E8F0` (중간 회색), `#94A3B8` (어두운 회색)
- **카드 배경**: `rgba(30, 41, 59, 0.5)` (반투명 다크)
- **테두리**: `rgba(148, 163, 184, 0.15)` (은은한 회색)

### 디자인 특징
- **그라데이션**: 파란색 계열 그라데이션 활발히 사용
- **글로우 효과**: `box-shadow`를 통한 부드러운 빛 효과
- **레이어 구조**: 배경 레이어, 컨텐츠 레이어의 명확한 구분
- **프리미엄 카드**: 반투명 배경 + 테두리 + 그림자
- **컴팩트 레이아웃**: 간결하고 효율적인 공간 활용

### 페이지 컨셉
- **"Control Board"**: 조종석/제어판 컨셉
- **"DATA ASSET CONTROL BOARD"**: 데이터 자산 구축 현황판
- **프리미엄 운영 OS**: 고급스러운 인터페이스

## 💡 제목 디자인 제안

### 옵션 1: 프리미엄 컨트롤 패널 스타일 (추천) ⭐

**컨셉**: Control Board의 메인 제어 패널처럼 보이게

**디자인 요소**:
```
┌─────────────────────────────────────────┐
│  ✍  데이터 입력 센터                     │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  매장 운영 OS 조종석                     │
└─────────────────────────────────────────┘
```

**스타일 특징**:
- **배경**: 반투명 다크 카드 (`rgba(30, 41, 59, 0.6)`)
- **테두리**: 상단에 파란색 네온 바 (`4px`, 그라데이션)
- **아이콘**: 더 큰 크기 (`2.5rem`), 파란색 글로우 효과
- **제목**: 큰 폰트 (`2.5rem`), 그라데이션 텍스트 (파란색 → 밝은 파란색)
- **부제목**: "매장 운영 OS 조종석" (작은 글씨, 회색)
- **하단 바**: 파란색 그라데이션 바 (현재 유지)
- **그림자**: 부드러운 글로우 효과

**CSS 예시**:
```css

    background: rgba(30, 41, 59, 0.6);

        transparent 0%,

        rgba(96, 165, 250, 1) 50%, 
        rgba(59, 130, 246, 0.8) 80%, 
        transparent 100%
    ) 1;
    border-radius: 12px;
    padding: 1.5rem 2rem;
    margin-bottom: 2rem;
    box-shadow: 
        0 0 30px rgba(59, 130, 246, 0.2),
        0 4px 20px rgba(0, 0, 0, 0.3);
}

.ps-page-header-title {
    font-size: 2.5rem;
    font-weight: 800;
    background: linear-gradient(135deg, #60A5FA 0%, #3B82F6 100%);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
    background-clip: text;
    text-shadow: 0 0 30px rgba(96, 165, 250, 0.5);
    letter-spacing: -0.03em;
}

    font-size: 2.5rem;
    filter: drop-shadow(0 0 10px rgba(96, 165, 250, 0.6));
}
```

### 옵션 2: 네온 사이버펑크 스타일

**컨셉**: 미래지향적이고 기술적인 느낌

**디자인 요소**:
- **배경**: 완전 투명 (배경 레이어 활용)
- **텍스트**: 네온 파란색 (`#00D9FF`), 글로우 효과 강화
- **아이콘**: 회전하는 글로우 효과
- **하단**: 애니메이션되는 네온 바
- **장식**: 좌우에 작은 네온 점들

**스타일 특징**:
- 매우 눈에 띄는 디자인
- 기술적이고 미래지향적인 느낌
- 다소 과할 수 있음

---

### 옵션 3: 미니멀 프리미엄 스타일

**컨셉**: 깔끔하고 세련된 고급스러움

**디자인 요소**:
- **배경**: 투명 또는 매우 약한 배경
- **제목**: 큰 폰트, 단순한 색상 (흰색 또는 파란색)
- **장식**: 최소한의 요소만 사용
- **간격**: 넓은 여백

**스타일 특징**:
- 가장 깔끔하고 안정적
- 페이지의 다른 요소와 자연스럽게 조화
- 다소 단조로울 수 있음

## 🎯 추천안: 옵션 4 (레이어드 카드 스타일)

### 추천 이유
1. **일관성**: 페이지의 카드 스타일과 완벽하게 일치
2. **균형**: 눈에 띄면서도 과하지 않음
3. **프리미엄**: 고급스러우면서도 실용적
4. **조화**: Control Board 컨셉과 자연스럽게 어울림
5. **확장성**: 나중에 추가 요소를 넣기 쉬움

### 주요 특징
- ✅ 페이지의 다른 카드들과 시각적 일관성
- ✅ 상단 네온 바로 프리미엄 느낌 강조
- ✅ 적절한 여백과 계층 구조
- ✅ 부제목으로 컨셉 명확화
- ✅ 호버 효과로 인터랙티브함 추가 가능

### 접근성
- 충분한 색상 대비 유지
- 텍스트 크기 적절히 설정
- 스크린 리더 호환성 유지

## 📝 참고사항

- 현재 페이지는 `data-ps-scope="input_hub"` 사용

- 또는 별도의 헤더 전용 CSS 함수 생성 가능
