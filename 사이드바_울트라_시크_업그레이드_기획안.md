# 사이드바 울트라 시크 업그레이드 기획안

## 📋 목표

**현재 프리미엄 블랙 테마를 "울트라 시크" 수준으로 한 단계 더 업그레이드**
- 사용자가 보는 순간 "와, 이거 진짜 멋있다"라고 느끼게
- 고급스러운 입체감과 깊이감
- 부드럽고 세련된 애니메이션
- 모던하고 시크한 디자인 언어

---

## ⚠️ 제약사항 유지

- JavaScript, DOM 조작 금지
- `[data-testid="stSidebar"]` 스코프만 사용 (위치/크기/토글 CSS 금지)
- 메인 콘텐츠에 영향 주는 전역 CSS 금지
- 모든 CSS는 `[data-testid="stSidebar"]` 하위로만
- `transform: translateX()` 금지 (레이아웃 흔들림 방지)
- CSS만으로 구현 (외부 리소스 없음)

---

## 🎨 울트라 시크 업그레이드 요소

### 1. 입체감 & 깊이감 (3D Depth)
- **레이어드 그림자**: 다층 그림자로 입체감 강화
- **미묘한 네온 글로우**: 버튼 주변 미묘한 빛 효과
- **그라데이션 오버레이**: 다층 그라데이션으로 깊이감
- **백드롭 분리**: 배경과 콘텐츠의 명확한 분리

### 2. 고급 애니메이션
- **부드러운 페이드 인**: 요소 등장 시 자연스러운 페이드
- **마이크로 인터랙션**: 호버 시 미묘한 반응
- **스무스 트랜지션**: 모든 상태 변화가 부드럽게
- **펄스 리듬**: 활성 버튼의 생동감 있는 펄스

### 3. 색상 & 그라데이션 고도화
- **다채로운 그라데이션**: 더 풍부한 색상 변화
- **미묘한 색상 전환**: 호버 시 자연스러운 색 변화
- **네온 악센트**: 파란색 계열의 미묘한 네온 효과
- **투명도 레이어링**: 여러 투명도 레이어로 깊이감

### 4. 타이포그래피 & 간격
- **더 넓은 여백**: 여유 있는 레이아웃
- **세련된 폰트 웨이트**: 계층감 있는 텍스트
- **미묘한 레터 스페이싱**: 고급스러운 간격

### 5. 버튼 높이 통일 (2줄 텍스트 대응)
- **고정 최소 높이**: 모든 버튼이 동일한 최소 높이 유지
- **일관된 패딩**: 텍스트 길이와 무관하게 동일한 패딩
- **텍스트 줄바꿈 처리**: 2줄 이상 텍스트도 깔끔하게 표시
- **정렬 및 간격**: 텍스트 정렬과 줄 간격 최적화

### 6. 시각적 계층
- **명확한 섹션 구분**: 카테고리 간 시각적 분리
- **포커스 강조**: 활성 요소의 명확한 강조
- **그룹핑**: 관련 요소들의 시각적 그룹화

### 1. CSS Keyframes 애니메이션 추가

```css
/* 부드러운 페이드 인 (요소 등장) */
@keyframes ultra-fade-in {
    from {
        opacity: 0;
        transform: translateY(-8px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

/* 마이크로 바운스 (호버 시 미묘한 반응) */
@keyframes ultra-micro-bounce {
    0%, 100% {
        transform: scale(1);
    }
    50% {
        transform: scale(1.02);
    }
}

/* 네온 글로우 펄스 (활성 버튼) */
@keyframes ultra-neon-pulse {
    0%, 100% {
        box-shadow: 0 4px 12px rgba(59, 130, 246, 0.3),
                    0 0 0 0 rgba(59, 130, 246, 0.4),
                    0 0 20px rgba(59, 130, 246, 0.1);
    }
    50% {
        box-shadow: 0 6px 24px rgba(59, 130, 246, 0.5),
                    0 0 0 4px rgba(59, 130, 246, 0.2),
                    0 0 30px rgba(59, 130, 246, 0.2);
    }

        background-position: 0% 50%;
    }
    50% {
        background-position: 100% 50%;
    }
```

## ⚠️ 주의사항

### 1. Transform 사용 최소화
- `scale(1.01)`만 사용 (레이아웃 영향 최소)
- `translateX`는 절대 사용 안 함
- 페이드 인에만 `translateY` 사용 (애니메이션 종료 후 원위치)

### 2. 성능 고려
- 애니메이션은 `transform`과 `opacity` 중심 (GPU 가속)
- 다층 그림자는 성능에 영향 가능 (최적화 필요시 조정)
- `will-change` 속성 사용 고려 (필요시)

### 3. 접근성
- 애니메이션은 시각적 향상용
- 기능에는 영향 없음
- `prefers-reduced-motion` 미디어 쿼리 유지

### 4. 브라우저 호환성
- 그라데이션 텍스트: `-webkit-background-clip: text` 사용 (fallback 제공)
- 백드롭 블러: `backdrop-filter` + `-webkit-backdrop-filter` 사용
- 다층 그림자: 모든 브라우저 지원

## 💡 버튼 높이 통일 상세 구현

### 문제점
- 버튼 텍스트가 2줄로 되면 버튼 크기가 제각각
- 1줄 버튼과 2줄 버튼의 높이가 달라 시각적 불일치
- 레이아웃이 어수선해 보임

### 해결 방안

#### 1. 최소 높이 고정
```css
[data-testid="stSidebar"] .stButton > button,
[data-testid="stSidebar"] button[kind],
[data-testid="stSidebar"] button {
    min-height: 3.5rem;  /* 2줄 텍스트 기준 최소 높이 */
}
```

#### 2. 수직 중앙 정렬
```css
display: flex;
align-items: center;
```
- 텍스트가 1줄이든 2줄이든 버튼 내부에서 수직 중앙 정렬
- `flex` 사용으로 자연스러운 정렬

#### 3. 텍스트 줄바꿈 처리
```css
word-wrap: break-word;
word-break: break-word;
white-space: normal;
line-height: 1.5;
```
- 긴 단어도 자동으로 줄바꿈
- 2줄 이상 텍스트도 깔끔하게 표시
- 줄 간격 최적화로 가독성 향상

#### 4. Expander 내부 버튼도 동일 적용
```css
[data-testid="stSidebar"] .stExpander .stButton > button,
[data-testid="stSidebar"] .stExpander button {
    min-height: 3.5rem;
    display: flex;
    align-items: center;
    line-height: 1.5;
    word-wrap: break-word;
    word-break: break-word;
    white-space: normal;
}
```

#### 5. 시스템 버튼도 동일 적용
```css
[data-testid="stSidebar"] .premium-system-section .stButton > button,
[data-testid="stSidebar"] .premium-system-section button {
    min-height: 3.5rem;
    display: flex;
    align-items: center;
    line-height: 1.5;
    word-wrap: break-word;
    word-break: break-word;
    white-space: normal;
}
```

### 효과
- ✅ 모든 버튼이 동일한 최소 높이 유지
- ✅ 1줄/2줄 텍스트 모두 깔끔하게 표시
- ✅ 시각적 일관성 확보
- ✅ 레이아웃 정돈
