## 1. 제품 한 장 요약 (One Pager)

### 한줄 정의
**"외식업 점장을 위한 숫자 기반 운영 코치 - 매일 입력하면 자동으로 문제를 찾아주고, 고칠 곳을 알려주는 시스템"**

### 타겟 유저
- **1차 타겟**: 소규모 외식업 점장 (1-3개 매장 운영)
  - 일일 매출/비용 수기 기록에 지친 점장
  - 원가 관리가 어려워 수익성이 불투명한 점장
  - 매출이 떨어졌을 때 원인을 모르는 점장

### 핵심 가치 제안
1. **자동 진단**: 매출/원가/구조 위험 자동 감지
2. **전략 자동 생성**: 가게 상태에 따른 TOP3 전략 카드 제공
3. **실행 유도**: "오늘 무엇을 해야 하는가" 중심의 운영 지시
4. **통합 입력**: 일일 입력을 하나의 화면으로 통합 (1분 안에 완료)

### 차별점
- **단순 입력 도구가 아님**: 입력 → 자동 판단 → 행동 유도
- **설계 중심**: 가게 구조(메뉴/재료/수익)를 설계하는 전략실 개념
- **코치 역할**: 숫자 중심이 아닌 "오늘 무엇을 해야 하는가" 중심

### 핵심 루프 (입력→판단→행동)

```
[매일] 일일 입력(통합)
    ↓
매출/방문자/판매량 입력 (1분)
    ↓
[자동] 홈에서 판단
    ↓
오늘의 운영 지시 생성
    - "오늘은 '마진 구조 복구'부터 하세요"
    - 근거: "손익분기점 대비 92%, 마진 메뉴 0개"
    ↓
[행동] CTA 버튼 클릭
    ↓
설계실/분석 페이지로 이동
    ↓
문제 해결 → 다음 날 입력 반복
```

**주간 루프**:
```
[주간] 문제 감지
    ↓
매출 하락 원인 찾기 (3단 플로우)
    ↓
원인 파악 → 설계실로 이동
    ↓
구조 개선 → 다음 주 확인
```

**월간 루프**:
```
[월말] 월간 성적표 확인
    ↓
실제 매출 vs 실제 비용 → 이익률 확인
    ↓
구조 리포트 확인
    ↓
다음 달 전략 수립
```

## 2. 사이드바 IA v1 최종안 (사장 사용 흐름 기준)

### 설계 원칙
1. **매일 쓰는 것 최상단**: 일일 입력 → 홈
2. **문제 해결 흐름**: 문제 분석 → 설계실
3. **설정은 최소화**: 마스터/목표는 필요 시에만
4. **리포트는 결과 확인**: 성적표/리포트는 하단

### 최종 IA 구조

```
🏠 HOME
├── 홈 (오늘의 운영 지시)
└── 전략 보드 (이번 달 전략)

🟢 오늘 입력
└── 일일 입력(통합)

🚨 문제 분석
├── 매출 하락 원인 찾기
├── 매출 분석
├── 판매·메뉴 분석
└── 원가 분석

🔥 가게 설계 (Expander)
├── 가게 설계 센터
├── 메뉴 포트폴리오 설계실
├── 메뉴 수익 구조 설계실
├── 재료 구조 설계실
└── 수익 구조 설계실

🛠 마스터·목표
├── 메뉴 관리
├── 재료 관리
├── 레시피 관리
├── 목표 비용 구조
└── 목표 매출 구조

📄 성적표·리포트
├── 월간 성적표
└── 주간 리포트

👥 운영 도구
├── 직원 연락망
├── 협력사 연락망
└── 게시판
```

### 카테고리별 역할 및 사용 빈도

| 카테고리 | 역할 | 사용 빈도 | 우선순위 |
|---------|------|----------|---------|
| 🏠 HOME | 진입점, 오늘의 지시 | 매일 | 최우선 |
| 🟢 오늘 입력 | 일일 데이터 입력 | 매일 | 최우선 |
| 🚨 문제 분석 | 원인 분석 및 진단 | 주 2-3회 | 높음 |
| 🔥 가게 설계 | 구조 설계 및 진단 | 주 1-2회 | 높음 |
| 🛠 마스터·목표 | 초기 설정/목표 입력 | 월 1-2회 | 중간 |
| 📄 성적표·리포트 | 결과 확인 | 월 1-2회 | 중간 |
| 👥 운영 도구 | 부가 기능 | 필요 시 | 낮음 |

### 변경 사항 (기존 대비)
- **"가게 설계"를 4번째로 이동**: 매일 쓰는 것 우선
- **"오늘 입력"을 2번째로 상승**: 매일 사용
- **"마스터·목표"로 통합**: 설정은 필요 시에만
- **"운영 도구" 최하단**: 부가 기능

## 4. 발주/재고 v1 스코프 정리

### v1에 포함
1. **재료 구조 설계** (재료 구조 설계실)
   - 원가 TOP3 집중도 파악
   - 고위험 재료 식별 (상위 원가 20% + 사용 메뉴 3개 이상)
   - 대체 가능 여부 설정 (DB 저장)
   - 발주 유형 설정 (단일/복수/미설정, DB 저장)
   - 안전재고 설정 (재료 관리에서)

2. **이론 소모 분석**
   - 판매량 + 레시피 기반 재료 사용량 계산
   - 기간별 재료 사용량 집계
   - 재료별 총 사용금액 계산
   - 재료 영향도 테이블 (연결 메뉴 수, 위험도)

### v1에서 제외 (보류)
1. **입고 관리**
   - 입고 기록 입력
   - 입고 → 재고 자동 반영
   - 입고 이력 관리

2. **실제 재고 관리**
   - 현재고 수동 입력 (안전재고 설정만 유지)
   - 재고 차이 분석
   - 재고 조정

3. **발주 실행**
   - 발주 추천 계산 (예상소요량 기반)
   - 발주 리스트 생성
   - 발주 CSV 다운로드

### 대안 (v1에서 제공)
- **재료 구조 설계실**에서:
  - "이론 소모 분석" 섹션 추가
  - 판매량 기반 재료 사용량 자동 계산
  - 재료별 사용금액 및 영향도 표시
  - 고위험 재료 식별 및 대체/발주 구조 설계

- **재료 관리**에서:
  - 안전재고 설정만 유지
  - 현재고 입력은 제거 (v2에서 입고 관리와 함께)

### 향후 확장 (v2+)
- 입고 관리 페이지 신규 생성
- 입고 → 재고 자동 반영 로직
- 발주 추천 계산 (이론 소모 + 안전재고 기반)
- 발주 실행 및 이력 관리

## 5. 사장 사용 흐름별 페이지 우선순위

### 매일 사용 (최우선)
1. **일일 입력(통합)** → 매출/방문자/판매량 입력 (1분)
2. **홈** → 오늘의 운영 지시 확인

### 주간 사용 (높음)
3. **매출 하락 원인 찾기** → 문제 감지 시 즉시 진단
4. **가게 설계 센터** → 주간 구조 점검
5. **설계실 4개** → 문제 해결 시 해당 설계실 이동
6. **매출 분석 / 판매·메뉴 분석** → 주간 성과 확인

### 월간 사용 (중간)
7. **월간 성적표** → 월말 정산 및 구조 리포트
8. **전략 보드** → 이번 달 전략 확인
9. **목표 비용 구조 / 목표 매출 구조** → 다음 달 목표 설정

### 필요 시 사용 (낮음)
10. **마스터 관리** → 메뉴/재료/레시피 등록/수정
11. **주간 리포트** → PDF 생성
12. **운영 도구** → 연락망, 게시판

## 6. v1 핵심 가치 체인

### 입력 → 판단 → 행동

```
[입력] 일일 입력(통합)
    ↓ 매출/방문자/판매량
[판단] 홈 (자동)
    ↓ 오늘의 운영 지시
[행동] CTA 버튼 클릭
    ↓ 설계실/분석 페이지
[해결] 문제 해결
    ↓ 다음 날 입력 반복
```

### 문제 감지 → 원인 파악 → 해결

```
[감지] 홈/설계 센터에서 위험 신호
    ↓
[원인] 매출 하락 원인 찾기
    ↓ 3단 플로우
[해결] 설계실로 이동
    ↓ 구조 개선
[확인] 다음 주 확인
```

### 설계 → 실행 → 모니터링

```
[설계] 설계실에서 구조 설계
    ↓ 역할 태그, 대체재, 발주 구조
[실행] 전략 보드에서 실행 로드맵
    ↓ 주간 TOP3 작업
[모니터링] 월간 성적표에서 확인
    ↓ 구조 리포트
[재설계] 필요 시 설계실로 복귀
```

## 7. v1 컷라인 정리

### 제외 기능 (v2+로 보류)
1. **입고 관리**: 입고 기록 → 재고 자동 반영
2. **실제 재고 관리**: 현재고 수동 입력, 재고 조정
3. **발주 실행**: 발주 추천 계산, 발주 리스트 생성
4. **방문·객단가 분석**: 별도 페이지 (매출 하락 원인 찾기에 통합)
5. **POS 연동**: 자동 매출 입력
6. **직원 관리**: 근무 스케줄, 인건비 자동 계산
7. **공급업체 관리**: 발주 업체 정보, 발주 이력

### 최소화 기능 (기능 유지, 노출 낮춤)
1. **점장 마감 / 매출보정 / 판매량보정**: 사이드바 제거, 직접 접근만
2. **재료 사용량 집계**: 재료 구조 설계실에 통합 가능
3. **운영 도구**: 기능 최소화, 하단 배치

### 강화 기능 (v1 핵심)
1. **홈**: 오늘의 운영 지시 최우선 표시
2. **일일 입력 통합**: 매일 1분 안에 완료 가능
3. **매출 하락 원인 찾기**: 3분 안에 원인 파악
4. **설계실 4개**: 구조 설계 및 진단
5. **전략 보드**: 이번 달 전략 + 주간 실행 로드맵
