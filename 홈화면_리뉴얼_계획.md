# 홈화면 디자인 리뉴얼 업그레이드 계획

## 📋 현재 상태 분석

### 현재 구조 (HOME v2)
1. **브랜드 히어로** - CAUSE OS, 슬로건, CTA 버튼
2. **오늘 상태 대시보드** - 미니 카드 3개 (마감/레시피/목표), 추천 메시지, 액션 버튼, 7일 요약
3. **STEP 1-3 카드** - 입력/분석/설계 각각 별도 카드

### 문제점
- 섹션이 너무 많아 복잡함 (3개 주요 섹션 + 각 STEP 카드)
- 정보 밀도가 높아 핵심 메시지가 희석됨
- 카드가 많아 시각적 피로감
- "어렵다"는 느낌 - 정보 과부하

## 🎯 리뉴얼 목표

### 핵심 원칙
1. **간결함** - 한 화면에 핵심만
2. **명확함** - 다음 행동이 바로 보임
3. **통합** - 관련 정보는 묶어서 표시
4. **계층** - 중요도에 따른 시각적 위계

- 스크롤 최소화


**장점:**
- 현재 구조 유지하면서 개선
- STEP을 압축하여 공간 절약

## 💡 추천안: [안 1] 통합형 2단 구조

### 구조 상세

#### SECTION 1: 브랜드 히어로 (기존 유지)
- CAUSE OS
- 슬로건
- [오늘 숫자 입력하기] CTA

#### SECTION 2: 오늘의 액션 센터 (통합)
하나의 큰 카드 안에 3개 블록:

**블록 1: 상태 요약**
- 미니 카드 3개 (마감/레시피/목표) - 기존 유지
- 더 작고 컴팩트하게

**블록 2: 오늘 추천**
- 추천 메시지
- 액션 버튼
- 7일 요약 (작게)

**블록 3: STEP 가이드 (압축)**
- 3단계를 한 줄 또는 2줄로 압축
- 각 STEP은 아이콘 + 제목 + 버튼만
- 설명 텍스트 최소화 또는 툴팁으로

## 🛠 구현 방향

### 1. HTML 구조 통합
```html
<div class="ps-action-center">
  <!-- 블록 1: 상태 요약 -->
  <div class="ps-status-summary">
    <!-- 미니 카드 3개 -->
  </div>
  
  <!-- 블록 2: 오늘 추천 -->
  <div class="ps-today-reco">
    <!-- 추천 메시지 + 버튼 -->
  </div>
  
  <!-- 블록 3: STEP 가이드 (압축) -->
  <div class="ps-step-guide-compact">
    <!-- 3단계를 한 줄 또는 2줄로 -->
  </div>
</div>
```

### 2. CSS 간소화
- 하나의 큰 카드로 통합
- 내부 블록은 구분선으로 분리
- STEP 카드의 긴 설명 제거 또는 최소화

### 3. STEP 압축 방법
**옵션 A: 한 줄 버튼 그룹**
```
[STEP 1: 입력] [STEP 2: 분석] [STEP 3: 설계]
```

**옵션 B: 2줄 그리드**
```
[STEP 1: 입력]  [STEP 2: 분석]
[STEP 3: 설계]  (넓게)
```

**옵션 C: 탭/토글**
- 기본은 버튼만 보이고, 클릭하면 설명 펼침
