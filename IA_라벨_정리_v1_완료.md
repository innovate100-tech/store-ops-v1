## 9️⃣ 주의 사항

### 9.1 Page Key 유지

**중요**: 모든 라우팅에서 page key는 기존 그대로 유지됩니다.

- 사이드바: 라벨만 변경, page key는 기존 유지
- 입력 허브: 버튼 텍스트만 변경, page key는 기존 유지
- 내부 이동: 모든 버튼에서 page key는 기존 유지

**이유**: 라우팅 로직(`elif page == "일일 입력(통합)"`)과의 호환성 유지

### 9.2 라벨 일관성

**규칙**:
- 입력 그룹: "입력" 접미사 통일
- 설계 그룹: "설계" 접미사 통일, "실" 제거
- 분석 그룹: "요약" 접미사 추가

**이유**: 사용자 이해도 향상 및 일관된 용어 사용
