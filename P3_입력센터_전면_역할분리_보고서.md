# P3: 입력센터 전면 역할 분리 리팩터링 보고서

## 작업 목표
입력센터의 모든 하위 페이지를 "입력 전용 화면"으로 전면 재정리
- 오직 숫자 입력, 저장, 확정만 수행
- 분석/판결/비교/전략/성적표/리포트/히스토리/PDF 제거

**제거한 섹션**:

```
│   ├── 1. 기간 선택 (연/월)
│   ├── 2. 목표 매출 입력
│   ├── 3. 입력 상태 확인 (손익분기점 숫자만)
│   └── 4. 비용 구조 입력
│       ├── 임차료
│       ├── 인건비
│       ├── 재료비
│       ├── 공과금
│       └── 부가세&카드수수료
└── Action Bar (하단)
    ├── Primary: 💾 목표 저장
    └── Secondary: 📋 전월 데이터 복사
```
